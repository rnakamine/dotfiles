# dotfiles

[![MIT License](http://img.shields.io/badge/license-MIT-blue.svg?style=flat)](LICENSE)

## Setup
```
mkdir -p ~/src/github.com/rnakamine/
cd ~/src/github.com/rnakamine/
git clone https://github.com/rnakamine/dotfiles.git
git submodule init
```

## Run script
```
./script/install.sh
./script/brew.sh
```
