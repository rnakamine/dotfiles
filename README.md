# dotfiles
rnakamine's dotfiles

## Initialize
1. Clone repository
```
mkdir -p ~/src/github.com/rnakamine/
cd ~/src/github.com/rnakamine/
git clone https://github.com/rnakamine/dotfiles.git
cd dotfiles/
```

2. Initialize git submodule
```
git submodule init && git submodule update
```

## Run setup script
```
./script/brew.sh
./script/install.sh
```
